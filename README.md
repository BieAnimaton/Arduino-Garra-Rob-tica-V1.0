## Components :point_up:
- Arduino Uno x1.
- Servo motor x4.
- 9V Duracell AA Battery (to Arduino) x1.
- 1,5V Duracell AA Battery x4.
- IRremote signal receiver x1.
- IRremote sender control x1.
- Body for Robotic Claw.
- Jumpers xN.
